## Overview

CRIX is a robust personal finance tool built to help users seamlessly manage their daily transactions, track expenses, and monitor their overall financial health. The application provides an intuitive interface for quick data entry and clear visualization of spending patterns.

## Features

- **Transaction Management:** Quickly log income and expenses with detailed categorization.
- **Financial Analytics:** Clear visual representations of spending trends and budget limits.
- **Secure Storage:** Fast, reliable local and cloud-synced storage for your financial data.
- **Modern Interface:** A clean, user-friendly design ensuring a smooth daily tracking experience.
